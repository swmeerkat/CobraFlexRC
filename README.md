# CobraFlexRC
## Getting Started
UCobraFlexRC is a JavaFX based application that provides a simple remote control of Waveshare 
Cobra Flex from a host computer via WLAN. It uses the web api (http GET) of the integrated ESP32 
controller.

* Host computer and CobraFlex must be in the same WLAN.
* The IP address of the CobraFlex is configured in application.properties.
